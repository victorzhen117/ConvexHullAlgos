Visual Convex Hull Algorithms

This is an application to help show a convex hull based on a series of points placed by a user. By definition, a convex hull is a polygon with smallest area that contains all the points. See below for more information about convex hulls and the algorithms used to make them.

To use this application, click on the canvas below to place points, and click "Jarvis March", "Graham Scan", or "Chan's Algorithm" to observe the algorithms.
